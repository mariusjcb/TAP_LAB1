Mac Requirements:
-
- macOS 10.13
- Xcode 9

Linux Requirements:
-
- **clang**
- Swift 4 compile

For more about Swift on Linux read the Swift.org > [Getting Started Guide](https://swift.org/getting-started/#installing-swift)

**Input:**
```
COINS:  1, 2, 20, 45
SUM:    41
```

**Output:**
```
1 x 1 RON
2 x 20 RON
```
